## ✨ Features

| Feature | Details |
|---|---|
| 📱 Mobile-First Design | Base styles for mobile, scaled up with `min-width` media queries |
| 🛏 6 Room Types | Each with image, description, features, pricing, and "Request This Room" button |
| 📋 Booking Form | Scrolls to form with room pre-selected on button tap |
| 💬 WhatsApp Integration | Pre-filled message sent directly to lodge number |
| 📧 Email Integration | Pre-filled subject + body sent to lodge email |
| 💰 Live Price Estimate | Auto-calculates total based on dates and room selected |
| 🖼 Vertical Auto-Scroll Gallery | CSS-only infinite scroll — no JavaScript |
| 🗺 Google Maps Embed | Real lodge location embedded |
| ⭐ Real Google Reviews | Pulled from actual lodge Google listing |
| 📍 Location Section | Address, drive times, contact cards |
| ❓ FAQ Accordion | 6 common questions with smooth open/close |
| 🚀 Single File | All HTML, CSS, and JS in one `aninas.html` file |
| 📵 No Frameworks | Pure HTML, CSS, and vanilla JavaScript — zero dependencies |

## 🚀 How to Run

No build tools. No installs. Just open it.

```bash
# Clone the repo
git clone https://github.com/apexscale-studio/aninas-lodge.git

# Open in browser
open aninas-lodge/aninas.html
```

Or drag `aninas.html` into any browser.

---

## 🌐 How to Deploy

### Netlify (Recommended — Free)
1. Go to [netlify.com](https://netlify.com)
2. Drag and drop the `aninas-lodge/` folder onto the dashboard
3. Live URL in 30 seconds

### cPanel / Shared Hosting
1. Upload `aninas.html` and `images/` folder to `public_html`
2. Done

### GitHub Pages
1. Push to a GitHub repo
2. Settings → Pages → Source: `main` branch
3. Live at `https://username.github.io/aninas-lodge`
